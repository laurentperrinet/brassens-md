Les raisons qui, ce soir, m'ont rendu nostalgique,\
Sont les moins nobles des raisons,\
Et j'aurais sans nul doute enterré cette histoire\
Si, pour renouveler un peu mon répertoire\
Je n'avais besoin de chansons.
